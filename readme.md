#### How to setup the project

1. Open your terminal

2. Go to the project directory
   . Install NPM packages

```sh
npm install
```

3. Launch the app

```sh
nodemon server.js
```

#### Built with

- NodeJS
- Express
- MongoDB + Mongoose
- jsonwebtoken
- Nodemon
- express-rate-limit
- bcrypt

<br/>
The .env file is used for the exercise but should never be public for production environnements
